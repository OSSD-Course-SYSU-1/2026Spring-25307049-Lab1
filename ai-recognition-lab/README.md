# AI文字识别示例（课程作业）
## 项目简介
本项目基于HarmonyOS ArkTS开发，实现相机拍摄+AI文字识别功能，是鸿蒙应用开发课程的示例作业。

## 核心功能
1.  相机拍摄：调用设备相机接口，拍摄带文字的图片
2.  AI识别：通过`textRecognition`接口，提取图片中的文字
3.  结果展示：识别完成后弹窗显示识别结果

## 项目结构
- `entry/src/main/ets/pages/Index.ets`：主页面代码，包含相机界面、文字识别逻辑
- `entry/src/main/module.json5`：应用配置，声明相机等权限
- `oh-package.json5`：依赖配置，包含相机、文字识别SDK

## 调试与修改记录
1.  编译运行：项目已在DevEco Studio中编译成功，可正常启动运行
2.  界面修改：修改了主界面文字，将“识别文字”改为“我的AI文字识别工具”，调整了字体大小和样式

## 运行说明
1.  使用DevEco Studio打开项目，执行`ohpm install`安装依赖
2.  运行到鸿蒙模拟器，APP会自动启动
3.  点击拍摄按钮，即可触发文字识别功能
